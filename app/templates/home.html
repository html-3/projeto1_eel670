Projeto 1 para EEL670

Este é nosso protótipo.

Para o primeiro projeto de Linguagens de Programação descidimos fazer um site web que visa coletar documentos como listas, provas e livros dos cursos de engenharia para disponibilizar-los a um público extenso. Entre as capacidades do nosso site é possivel:

1. Capacidade de adicionar documentos pelos usuários
2. Necessidade dum cadastro com confirmação por email.
3. Logins únicos para usuários e administradores.
4. Edição e remoção de documentos de forma prática.
